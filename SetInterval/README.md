# Lua Interval Example

This project demonstrates how to create a simple `setInterval` function in Lua using the `luatime` module.  
The interval repeatedly runs a given function after a specified delay.

## How It Works

The `setInterval(func, delay)` function:

- Takes a function `func`
- Takes a delay `delay` in seconds
- Runs the function over and over with the given delay

The `hello` function prints numbers from 1 to 100.
