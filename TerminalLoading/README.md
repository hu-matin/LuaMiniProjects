# Lua Spinner

A lightweight terminal loading animation written in Lua. Creates a spinning indicator (|/-\) for progress display.

## What It Does

- Shows animated spinner while scripts run  
- Uses carriage return (\r) and flush for smooth animation  
- Customizable speed and characters  
- Works on Windows/Linux/Mac terminals  

## Requirements

- Lua 5.1+ and ANSI-compatible terminal
- **Luatime library** [download](https://github.com/hu-matin/luatime)
